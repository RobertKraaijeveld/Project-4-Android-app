Project 4 - App

===============

De folder /MainProjectCode is onze up-to-date source code.

Alle activities staan binnen de sub-package /Activities

Alle helper-classes die bv. netwerkinformatie afhandelen staan binnen de sub-package helpers.

Veel van deze helper classes maken gebruik van php scripts op onze webserver.
Deze scripts kunt u vinden in /DatabaseScripts*

*= Wij kunnen niet garanderen dat deze offline scripts 100% overeenkomen
met de scripts die wij online hebben staan.
